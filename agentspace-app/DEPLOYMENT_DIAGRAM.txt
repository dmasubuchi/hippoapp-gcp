+-------------------+      +-------------------+      +-------------------+
|                   |      |                   |      |                   |
|  ローカル開発環境   |----->|  Docker コンテナ   |----->|  Google Cloud Run  |
|                   |      |                   |      |                   |
+-------------------+      +-------------------+      +-------------------+
        |                          |                          |
        v                          v                          v
+-------------------+      +-------------------+      +-------------------+
|                   |      |                   |      |                   |
|  アプリケーション   |      |  コンテナイメージ  |      |  デプロイされた    |
|  コード            |      |                   |      |  アプリケーション   |
|                   |      |                   |      |                   |
+-------------------+      +-------------------+      +-------------------+
        |                          |                          |
        v                          v                          v
+-------------------+      +-------------------+      +-------------------+
|                   |      |                   |      |                   |
|  ローカルテスト    |      |  Container        |      |  公開URL          |
|                   |      |  Registry         |      |                   |
+-------------------+      +-------------------+      +-------------------+

## デプロイメントステップ

1. ローカル開発環境でアプリケーションコードを作成・テスト
2. Dockerfileを使用してコンテナイメージをビルド
3. Google Container Registryにイメージをプッシュ
4. Google Cloud Runにデプロイ
5. 公開URLでアプリケーションにアクセス
